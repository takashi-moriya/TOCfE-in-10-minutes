# TOCfE-in-10-minutes

This repository contains slides for TOCfE in 10 minutes, created using Markdown and Marp.

## How to View Slides

The slides are automatically built and deployed to GitHub Pages on every push to the main branch.

Visit [https://your-username.github.io/TOCfE-in-10-minutes/](https://your-username.github.io/TOCfE-in-10-minutes/) to view the slides.

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build slides:
   ```bash
   npm run build
   ```

3. Open `slides.html` in your browser.

## Editing Slides

Edit `slides.md` to modify the slides. Use Marp syntax for slide separation (---).
